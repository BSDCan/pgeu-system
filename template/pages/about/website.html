About the website

The design of this website is mainly copied from www.postgresql.org. The code is a simple wrapper around the django templating framework. Dynamic parts will be done using this framework if/when they appear.

The full source code for this website is available in our GIT repository under the BSD licence.

The hardware running the website is maintained as part of the postgresql.org infrastructure.
