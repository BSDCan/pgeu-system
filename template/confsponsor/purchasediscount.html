Purchase discount code - {{conference}}

On this page you can purchase a discount code for {{conference}}. An invoice for this discount code will be generate once it has been used fully, or once it has expired (based on the availability date). By signing up for a discount code, you undertake to pay the discounted part on this invoice, which will be made due for immediate payment once generated.

Discount codes generated here will not be published by the organizers, and the organizers take no responsibility for limiting the number of uses, other than that specified in the order. It is the responsibility of the organization ordering the discount codes to make sure they only reach the indented recipients.

A discount code can be generated for a fixed amount or a percentage, but not both (without generating separate discount codes). Each code will have a maximum number of uses, to limit the maximum total amount, as well as a final date for usage. Finally, a discount code can be set up to require a specific option to be ordered before it's valid, for example to limit the code only to attendees of a specific training session.
